지구과학 II: 진앙 위치 찾기

2023년 튀르키예-시리아 대지진 (Mw 7.8) 관측 자료 활용

1. 지진파 속도 설정

P파와 S파의 평균 속도를 입력하세요. (단위: km/s)

2. 관측소 데이터 및 진원 거리 계산

각 관측소의 PS시를 초 단위로 입력하면 진원 거리가 자동 계산됩니다. 체크박스를 통해 지도 표시 여부를 켜고 끌 수 있습니다.

표시 관측소 P파 도달 S파 도달 PS시 (초) 입력 진원 거리 (km)

3. 진앙 분석 도구

지도에서 원이 교차하는 두 점을 클릭하여 공통현을 그리세요.

선택된 교점 수: 0 / 2

교점 2개를 먼저 선택해주세요.

4. 진원 깊이 분석

지도에서 관측소를 클릭한 후, 찾은 진앙(별표)을 클릭하여 진원 깊이를 시각적으로 계산하세요.

추정된 진앙 위치

* 실제 튀르키예 지진 진앙: 37.174°N, 37.032°E
